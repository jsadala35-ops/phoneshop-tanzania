# Module Guide - PhoneShop Pro Edition

Detailed documentation for each module in PhoneShop Pro Edition.

## 1. Dashboard (Dashbodi)

### Purpose
Provide real-time business metrics and key performance indicators.

### Features
- **Device Count**: Total phones in inventory
- **Revenue Metrics**: Total and monthly revenue
- **Sales Summary**: Number of transactions
- **Customer Analytics**: Total customers and activity
- **Top Brands**: Best-selling phone brands
- **Performance Charts**: Visual representation of trends

### Key Metrics
- Total Inventory Value
- Monthly Revenue
- Sales Count (MTD)
- New Customers (MTD)
- Average Transaction Value
- Return Rate

### Usage
1. Access Dashboard from main menu
2. View summary cards for quick insights
3. Drill down into detailed charts
4. Export reports as PDF/Excel

## 2. Inventory Management (Hifadhi)

### Purpose
Manage phone inventory with detailed tracking and organization.

### Features
- **Brand Management**: Organize phones by brand
- **Model Tracking**: Track specific phone models
- **IMEI Registration**: Store IMEI numbers for warranty
- **Stock Levels**: Monitor inventory quantities
- **Price Management**: Set and update prices
- **Low Stock Alerts**: Notifications for low stock

### Data Fields
- Brand (Apple, Samsung, Huawei, etc.)
- Model (iPhone 13 Pro, Galaxy S21, etc.)
- IMEI (International Mobile Equipment Identity)
- Quantity in Stock
- Purchase Price (Cost)
- Selling Price
- Date Added
- Supplier
- Status (Available, Sold, Reserved)

### Workflows

**Add New Device:**
1. Navigate to Hifadhi > Add Device
2. Fill in device details
3. Enter IMEI number
4. Set prices
5. Save

**Update Stock:**
1. Find device in inventory
2. Click Edit
3. Adjust quantity
4. Update reason
5. Save

**Search & Filter:**
- By brand
- By model
- By IMEI
- By price range
- By stock status

## 3. Sales Management (Mauzo)

### Purpose
Track and manage all sales transactions.

### Features
- **Point of Sale**: Quick transaction entry
- **Receipt Generation**: Print/email receipts
- **Payment Methods**: Support multiple payment types
- **Discount Management**: Apply discounts and promotions
- **Transaction History**: Complete sales records
- **Refund Processing**: Handle refunds

### Payment Methods
- Cash (Pesa Taslimu)
- Mobile Money (M-Pesa, AirtelMoney)
- Bank Transfer
- Check
- Credit/Debit Card

### Process
1. Select customer or create new
2. Add devices to transaction
3. Apply discounts (if applicable)
4. Select payment method
5. Complete transaction
6. Print/email receipt

## 4. Returns & Exchanges (Kurundisha)

### Purpose
Manage product returns and exchanges efficiently.

### Features
- **Return Authorization**: Create return requests
- **Exchange Management**: Track device exchanges
- **Refund Processing**: Handle refunds
- **Warranty Claims**: Track warranty-related returns
- **Return History**: Complete return records

### Return Reasons
- Defective Unit
- Wrong Item Received
- Customer Changed Mind
- Hardware Issue
- Battery Problem
- Screen Damage
- Other (with description)

### Return Workflow
1. Find original transaction
2. Create return request
3. Specify reason and condition
4. Arrange replacement or refund
5. Process transaction
6. Update inventory

## 5. Customer Management (Watumiaji)

### Purpose
Maintain customer database and track relationships.

### Features
- **Customer Profiles**: Store customer information
- **Contact Management**: Phone, email, address
- **Purchase History**: Track all purchases
- **Spending Analytics**: Customer value analysis
- **Communication**: Send messages/notifications
- **Loyalty Program**: Track loyalty points (if enabled)

### Customer Fields
- Name
- Phone Number
- Email Address
- Location/Address
- ID Type (National ID, Passport)
- ID Number
- Date of Birth
- Gender
- Notes

### Customer Segments
- **New Customers**: Last purchase >90 days ago
- **Regular Customers**: 2+ purchases
- **VIP Customers**: High lifetime value
- **Inactive**: No purchase >6 months

## 7. User Management (Usajili)

### Purpose
Manage system users and permissions.

### User Roles

#### Admin
- Full system access
- User management
- System settings
- Report access

#### Manager
- View all reports
- Approve transactions
- Manage staff
- Limited settings access

#### Staff
- Process sales
- Manage inventory
- View own transactions
- Limited reports

#### Viewer
- Read-only access
- View reports
- No transaction capability

### User Management
1. Navigate to Usajili
2. Click Add User
3. Fill user details
4. Assign role
5. Set permissions
6. Send welcome email
